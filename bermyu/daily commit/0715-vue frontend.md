Composition API

- 여러 파일에 나뉜 코드를 일일이 확인해야 하는 어려움
- [공식문서](https://v3.ko.vuejs.org/guide)



로그인 기능을 담당했다고 했을 때, 실제 면접에서 과정을 자신있게 대답할 수 있는가?

내가 담당하는 부분이 아니더라도 프로세스 정도는 기억하고 있어야 한다.

로그인 마스터 vs 기능 구현

- 내 서비스 - 구글 로그인 클릭 - 구글 로그인 페이지 이동 - 구글 로그인 - access token 발급 - redirect URL - 원하는 회원 정보 - access token 전달 - 정보 전달
- 프론트는 보안이 약하다 (CORS 에러를 만든 이유)



UID, 가입 날짜, 프로필 사진, ... 컬럼을 정의해둘 것 (DB에 Insert할 것)

프로필 사진이 변경되는 것을 감지하고 프로필을 업데이트하는 등의 로직을 추가해줘야 함.

e.g. 닉네임 변경, 아이디 기억하기, 회원 탈퇴 시 같이 탈퇴, 비밀번호 찾기/재설정,
